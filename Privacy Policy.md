# Privacy Policy for Sun NXT Enhanced Player

**Effective Date:** January 18, 2026  
**Last Updated:** January 18, 2026

## Introduction

Sun NXT Enhanced Team ("we," "our," or "us") operates the Sun NXT Enhanced Player browser extension (the "Extension"). This Privacy Policy explains our practices regarding the collection, use, and disclosure of information when you use our Extension.

**Your privacy is important to us.** We are committed to protecting your personal information and your right to privacy.

## Summary

**TL;DR:** The Sun NXT Enhanced Player extension does NOT collect, store, transmit, or share any personal data. It operates entirely locally in your browser and only modifies the visual presentation of the Sun NXT website.

## Information Collection and Use

### We Do NOT Collect Any Data

The Sun NXT Enhanced Player extension is designed with privacy as a core principle. We do not collect, store, process, or transmit any personal or usage data whatsoever.

Specifically, we do NOT collect:
- Personal identification information (name, email, phone number, etc.)
- Browsing history or activity
- Video viewing habits or preferences
- IP addresses or device identifiers
- Cookies or tracking data
- Analytics or usage statistics
- Keyboard inputs or form data
- Location information
- Any other personal or non-personal information

### How the Extension Works

The Extension operates entirely within your browser and performs the following functions locally:

1. **DOM Manipulation**: Modifies the visual layout of the Sun NXT website to enable theatre mode
2. **Style Application**: Applies CSS styling to video player elements for fullscreen viewing
3. **UI Enhancement**: Adds close buttons and status indicators for better user experience
4. **Local State Management**: Stores theatre mode preferences in browser memory (not persistent)

All operations occur locally on your device and are not transmitted to any external servers.

## Permissions Explained

The Extension requests the following permissions, which are necessary for its functionality:

### 1. `activeTab`
**Purpose:** Allows the Extension to interact with the currently active tab when you explicitly invoke it.

**What it does:** Enables the Extension to modify the Sun NXT page layout when you activate theatre mode.

**What it does NOT do:** Does not allow access to other tabs, browsing history, or background tracking.

### 2. `scripting`
**Purpose:** Allows the Extension to inject content scripts into Sun NXT web pages.

**What it does:** Enables the Extension to run JavaScript code that modifies the player layout and controls.

**What it does NOT do:** Does not execute on any other websites or collect data from Sun NXT.

### 3. Host Permission: `https://www.sunnxt.com/*`
**Purpose:** Restricts the Extension to only work on Sun NXT website.

**What it does:** Allows the Extension to activate and function only on Sun NXT pages.

**What it does NOT do:** Does not access any other websites or domains.

## Data Storage

### Local Storage Only
The Extension uses browser's local memory to:
- Remember the current state of theatre mode (active/inactive) during the current browsing session
- Store original CSS styles to restore them when exiting theatre mode

**Important:** This data is:
- Stored only in your browser's memory (RAM)
- NOT persistent across browser sessions
- NOT transmitted to any servers
- NOT accessible to us or any third parties
- Automatically cleared when you close the tab or browser

### No Server Communication
The Extension does not communicate with any external servers, APIs, or services. All functionality is self-contained within the Extension code running in your browser.

### How to Uninstall

If you wish to remove the Extension:
1. Navigate to `chrome://extensions/` in your browser
2. Find "Sun NXT Enhanced Player"
3. Click "Remove"
4. Confirm removal

Uninstalling the Extension immediately removes all code and local state from your browser.
